# Dessert Dream

Односторінковий сайт кондитерської з авторськими тортами. Реалізовано на чистих HTML, CSS та JavaScript.

## Запуск

Відкрийте `index.html` у браузері або запустіть будь-який простий локальний сервер.

## Структура

- `index.html` — вся структура сторінки та контент
- `css/styles.css` — дизайн-система, адаптивність, анімації та стани елементів
- `js/script.js` — анімації появи секцій
- `images/` — папка для локальних зображень проєкту
